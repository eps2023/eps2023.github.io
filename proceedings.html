Submission procedure
A strong filename convention is required in order to be process by our automated paper procedure to build the final proceedings to be included in the 49th European Conference On Plasma Physics Proceedings. The file name should strictly follow the syntax {M,T,W,T}_{MCF,BIF,TDF,PSM}_{[0-9][0-9][0-9]}.pdf. So the files should be named for example M_SPSF_001.pdf. The only file format accepted is PDF.



** Warning ** Any file with a different naming convention won't be accepted..
